# claude-utils

A collection of utilities for [Claude Code](https://claude.ai/code).

## Tools

- [statusline](statusline/README.md) — Status line showing cwd, model, context usage, rate limits, and session cost.

## Skills

- [commit-message](skills/commit-message/README.md) — Suggest a commit message from the session's work.

See each link for install and usage.
